# Glossa Landing Page

This folder contains a dependency-free static landing page for Glossa.

Open `index.html` directly from the repository root, or serve the repository root with any static file server so the page can reference:

- `../Assets/Glossa-AppIcon.png`
- `../Assets/Glossa-MarkTemplate.png`

The page is intentionally plain HTML and CSS so it can be deployed later to GitHub Pages, Vercel, Netlify, or any static host without adding a JavaScript build pipeline.

Public download buttons point to the GitHub releases page. The local release ZIP is generated by `../script/package_release.sh` and is intentionally kept out of git under `dist/`.
